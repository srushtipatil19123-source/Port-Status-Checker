# Port Status Checker

A Python-based Port Status Checker that uses socket programming to determine whether a specific port on a given host or IP address is open or closed.

## Features
- Accepts user input for host/IP address
- Accepts user input for port number
- Checks port availability using Python's socket module
- Displays whether the port is OPEN or CLOSED

## Technologies Used
- Python 3
- Socket Programming

## Example

Enter Host/IP Address: 8.8.8.8

Enter Port Number: 80

Port 80 is OPEN on google.com
